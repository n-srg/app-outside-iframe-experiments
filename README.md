# How to use

  - Clone the repo
  - Run `npm install`
  - Bundle `Apps` applications: go to `Apps` repo and execute `npm run build`
  - Copy `build` folder to my repo's root
  - Rename `build` >>> `apps`
  - Execute `npm start`
  - Go to `http://localhost:3000`
